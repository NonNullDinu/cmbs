# Repository Structure

```tree
.
├── doc             // The documentation
├── libleaf         // The source code of the library
├── libleafcore     // A crate with code required to write extensions for other build systems
├── libutils        // A crate with some code useful for writing C/C++ meta build systems
├── src             // The source code of the executable
└── tools           // Some miscellaneous utils
```
